ChatConnect - MERN Stack Chat Application
🚀 Overview

ChatConnect is a full-stack real-time chat application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The platform enables users to create accounts, search for other users, add friends, and communicate through a clean and responsive chat interface.

The project focuses on user authentication, friend management, and seamless messaging while following modern web development practices.
